# Features

Domain-specific modules organized by feature area.

## Structure

Each feature folder should contain:
- Components specific to that feature
- Hooks and utilities for that feature
- Types specific to that feature

## Planned Features

- `customers/` - Customer profile management
- `contracts/` - Contract management
- `time-entries/` - Time logging
- `priorities/` - Priority tracking
- `requests/` - Customer request management
- `reporting/` - Reports and analytics
- `admin/` - Admin settings and reference data
