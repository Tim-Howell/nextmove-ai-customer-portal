## MODIFIED Requirements

### Requirement: Customer dashboard view
The system SHALL display real contract and time data on customer dashboard.

#### Scenario: Customer sees contracts count
- **WHEN** customer user views dashboard
- **THEN** system displays count of active contracts for their customer

#### Scenario: Customer sees hours summary
- **WHEN** customer user views dashboard
- **THEN** system displays total hours used this month across all contracts

#### Scenario: Customer sees recent time entries
- **WHEN** customer user views dashboard
- **THEN** system displays 5 most recent time entries for their customer
