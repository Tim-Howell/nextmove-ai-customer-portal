# Phase 15 - Portal Enhancements

## Why
The portal needs several usability improvements and new features to better serve both internal staff and customers. This phase addresses redundancy in navigation, enhances reporting capabilities, improves data entry workflows, and creates a more polished customer experience.

## What

### Navigation & UI Cleanup
- Remove Time Logs from customer navigation (redundant with Reports)
- Rename "Reports" to "Time Reports"

### Enhanced Reporting
- Multi-select category filter
- Multi-select contract filter
- Billable filter (Yes/No)
- Staff representative filter (multi-select)

### Time Entry Improvements
- Create default "On-Demand / Off Contract" contract for all customers
- Make contract a required field on time entry
- Allow staff to enter time on behalf of other staff members

### Internal Notes
- Add internal notes to customers (like requests)
- Add internal notes to priorities (like requests)
- Add internal notes to time logs (like requests)

### User & Profile Management
- Make system users editable (first name, last name, email, title, organization)
- Show all users in settings (not just current user)
- Allow editing of customer user profiles

### Portal Branding
- Create portal profile/company settings for NextMove AI
- Store: organization name, website, logo, description
- Display portal branding where appropriate

### Visual Enhancements
- Add square images to customers (logo/avatar)
- Add square images to priorities (visual identifier)

### Customer Dashboard Redesign
- Create dedicated customer dashboard layout
- Show: customer logo, company info, NextMove AI contacts
- Show: active priorities, active contracts
- Include current dashboard stats (excluding individual time logs)
- Replace current customer dashboard view
