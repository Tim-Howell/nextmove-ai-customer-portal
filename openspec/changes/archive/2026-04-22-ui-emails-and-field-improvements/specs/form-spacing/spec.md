## ADDED Requirements

### Requirement: Demo data checkbox spacing
The system SHALL display proper spacing above the demo data checkbox section on all create/edit forms.

#### Scenario: Spacing on customer form
- **WHEN** admin views the create or edit customer form
- **THEN** the demo data checkbox SHALL have adequate top margin separating it from the fields above

#### Scenario: Spacing on contact form
- **WHEN** admin views the create or edit contact form
- **THEN** the demo data checkbox SHALL have adequate top margin separating it from the fields above

#### Scenario: Spacing on contract form
- **WHEN** admin views the create or edit contract form
- **THEN** the demo data checkbox SHALL have adequate top margin separating it from the fields above

#### Scenario: Spacing on priority form
- **WHEN** admin views the create or edit priority form
- **THEN** the demo data checkbox SHALL have adequate top margin separating it from the fields above

#### Scenario: Spacing on request form
- **WHEN** admin views the create or edit request form
- **THEN** the demo data checkbox SHALL have adequate top margin separating it from the fields above

#### Scenario: Spacing on time entry form
- **WHEN** admin views the create or edit time entry form
- **THEN** the demo data checkbox SHALL have adequate top margin separating it from the fields above

#### Scenario: Consistent spacing across forms
- **WHEN** viewing any form with demo data checkbox
- **THEN** the spacing SHALL be consistent across all forms (same margin value)
