# Archived documentation

Point-in-time documents kept for historical reference. Not accurate for the
current state of the portal — issues called out in these reports have been
addressed in subsequent changes (see `openspec/changes/archive/`).

| File | Original purpose |
|---|---|
| `admin-portal-test-notes-2026-04.md` | Working notes captured while testing the admin portal pre-launch (April 2026). |
| `admin-portal-test-report-2026-04.md` | Formal admin-portal test report from the same testing pass. |
| `customer-portal-test-notes-2026-04.md` | Working notes captured while testing the customer portal pre-launch. |
| `customer-portal-evaluation-report-2026-04.md` | Formal customer-portal evaluation written by an external test pass. |

If you need to reconstruct what the portal looked like at go-live, these
documents plus the matching commit history are the source of truth.
